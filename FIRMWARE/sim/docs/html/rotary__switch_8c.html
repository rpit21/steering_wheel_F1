Detailed Description

Implementation of the rotary switch driver.

This file contains the logic for reading, calibrating, and interpreting the analog signal from a rotary switch connected to an ADC channel.

The driver performs a linear conversion between raw ADC values and discrete switch positions, providing a stable and calibrated interface for upper application layers.

The implementation depends on the HAL ADC module for raw value acquisition.
