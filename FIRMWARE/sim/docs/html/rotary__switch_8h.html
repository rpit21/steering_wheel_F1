Detailed Description

Driver interface for rotary switch position detection using ADC.

This module provides an interface to read and process analog input values from a rotary switch. The switch output is connected to an ADC channel and converted into discrete position indices (e.g., 0–7 or 0–15).

It supports calibration for minimum and maximum raw ADC values, allowing the driver to operate correctly despite component or voltage variations.

Note
Before using rotary_GetPosition(), call rotary_Init() and optionally rotary_SetCalibration(min, max) to set the input range.
