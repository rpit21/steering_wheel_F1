Subsystem ADC - Step-by-Step Guide

Layered Architecture

The ADC system is divided in the next layer:

  1. **HAL (hal\hal_adc*):** Manage the hardware layer (s32k) or create the simulated environment (host PC)
  2. Drivers: Have all the functions for processing the ADC signal for specific peripheral. You will find 2 drivers: clutch and rotary_switch
  3. **Test (test\adc_test.c):** Contains all high level code for testing the drivers and hal
  4. **adc_data.csv (test\adc_data.csv):** Contains an array of data that simulate the values given by an ADC. Each column represents a CHANEL and are separate by (,)

HAL APIs (hal\adc.h)

  • void hal_adc_init(void); Initialization and Configuration the ADC (s32k) or Setup for the simulation (host pc)
  • uint16_t hal_adc_read(uint8_t channel); Creates and return a 12 bits integer of the reading value of a specific channel

‍On host PC the values returned by the hal_adc_read are given by reading the adc_dat.cvs on the test/ folder. If you dont have any .cvs it would generate random values

Driver APIs

Clutch's Driver (clutch.h)

  • void clutch_Init(void); Initialization all internal variables and set a default calibration with a minimum value = 0 and a maximum value = 4095
  • void clutch_SetCalibration(uint16_t min, uint16_t max); Function to set a calibration by passing the minimum and maximum value of the analog input signal
  • uint16_t clutch_GetRawValue(void); Get the raw value from ADC
  • float clutch_GetPercentage(void); Transform the Raw ADC values into percentages.

Rotary Switch's Driver (rotary_switch.h)

  • void rotary_Init(uint8_t num_positions); Initilization all internal variables and set a default calibration with a minimum value = 0 and a maximum value = 4095. Here you insert the number of positions that you want
  • void rotary_SetCalibration(uint16_t min, uint16_t max); Function to set a calibration by passing the minimum and maximum value of the analog input signal
  • uint16_t rotary_GetRawValue(void); Get the raw value from ADC
  • uint8_t rotary_GetPosition(void); Transform the Raw ADC values into positions.

‍On every .h you will find ROTARY_ADC_CHANNEL and CLUTCH_ADC_CHANNEL which represents the corresponding channel where the rotary switch and clutch are connected.

Example of Use

Clutch

#include "../hal/hal_adc.h"
#include "../drivers/clutch.h"
#include <stdio.h>
void main(void) {
hal_adc_init(); //Initialization of the ADC
clutch_Init(); //Initialization of the clutch
clutch_SetCalibration(400,4000); // Putting min and max for calibration (optional)
while(1) {
uint16_t raw_c=clutch_GetRawValue(); //read and update the raw value (optional)
float clutch=clutch_GetPercentage(); // obtaing the porcentage value
printf("Clutch: %.1f%% | Raw value: %u\n", clutch, raw_c);
}
}
