Detailed Description

Steering Wheel status message structure.

This structure represents the data periodically transmitted from the Steering Wheel ECU over the CAN network. It contains:

  • Button states (4 bits)
  • Rotary switch position (0–15)
  • Clutch pedal position (0–100%)
